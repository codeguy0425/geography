# Geography

This is not a codebase. It is a study aid for an **S1 Geography Final Examination** (Hong Kong curriculum, English-medium).

## Content

| File | What it is |
|------|------------|
| `final.md` | Exam specification: date (23-Jun-2026, 60min), sections (A MC, B T/F, C structured, D bonus), coverage (C1 Units 4–6, map reading pp.70–79, E7 Tourism Units 1–2), required equipment |
| `final.jpeg` | Scanned exam paper or answer key |
| `book/book.md` | Study guide ToC: Unit 4 (urban problems), Unit 5 (solutions), Unit 6 (sustainable city) |
| `book/unit5.jpg`, `unit6.jpg`, `unit7.jpg` | Scanned textbook pages |

## No code

There is no build, test, lint, CI, or package manager. No git repo. No instruction files existed before this one. Treat all edits as plain markdown or image replacements.
